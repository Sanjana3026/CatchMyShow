create a react project 
    1. run "npm create vite@latest" in the project folder
    2. give the project name : client
    3. framework : react and variant : Javascript
    4. cd to client folder(craeted above) and run 
        "npm i"
        "npm i react-router-dom" -> helps navigate pages 
        "npm i lucide-react" -> icons
        "npm i react-hot-toast" -> notifications
        "npm i react-player" -> play videos 
    5. run " npm run dev " to run the project
    6. INSTALL TAILWIND CSS VIA "npm install tailwindcss @tailwindcss/vite"
